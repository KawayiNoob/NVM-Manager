# NVM Manager

一个轻量级的 nvm 桌面管理应用，使用 Electron 构建。

## 功能

- 显示当前 Node.js 版本
- 查看已安装的 Node.js 版本
- 切换 Node.js 版本
- 查看可下载的 Node.js 版本
- 安装新的 Node.js 版本

## 前置要求

- 已安装 nvm (Node Version Manager)
  - Windows: [nvm-windows](https://github.com/coreybutler/nvm-windows)
  - macOS/Linux: [nvm](https://github.com/nvm-sh/nvm)
- Node.js 和 npm

## 安装

```bash
npm install
```

## 运行

```bash
npm start
```

开发模式（打开开发者工具）：

```bash
npm run dev
```

## 使用说明

1. **已安装** - 查看所有已安装的 Node.js 版本，可以切换使用
2. **可下载** - 查看所有可用的 Node.js 版本，可以安装新的版本
3. **刷新** - 重新加载版本列表
